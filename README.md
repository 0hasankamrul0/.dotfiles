## Linux Settings
Clone The Repository. Go to the .dotfiles folder. Run clean_install with $HOME parameter
```bash
git clone https://github.com/0hasankamrul0/.dotfiles.git
cd .dotfiles
sudo bash clean_install $HOME
```
